MusicGame

　ゲーセンにあるアレを真似した音ゲーもどきです。 　もどきなので音は無いです。

____________________________

操作方法

　最初に難易度を選択

　　easyはホントに簡単なんであんまりオススメできないです(笑)

　•AかJキーで左端をたたく

　•SかKキーで左から２番目をたたく

　•DかLキーで右から２番目をたたく

　•Fか：キーで右端をたたく
